# RembrandtPhotoBooth

# Installation

## Backend

Installation on mac or linux

```
cd backend/
python3.11 -m venv venv
source venv/bin/activate
pip install "fastapi[all]"
pip install -r requirements.txt
```
## Frontend

```
npm install
```

## Execution

```
uvicorn main:app --reload
```
